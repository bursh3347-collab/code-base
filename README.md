## 🚀 Quick Start

1. **Browse modules** — Each folder is an independent module with its own README
2. **Read the README** — Understand what's inside and the recommended approach
3. **Copy what you need** — Every module is designed for copy-paste reuse
4. **Check SOURCES.md** — Verify licenses before using in your project

## 🏗️ Architecture Philosophy

- **Loose coupling** — Each module is independent, no cross-module dependencies
- **Copy-friendly** — Designed for file-level copy, not package management
- **Multiple options** — Each module offers 2-3 approaches with a clear recommendation
- **Source-traced** — Every code snippet links back to its origin project
